# Bridge Perf Analyser

## Overview
Tool to parse, analyze, and plot Cohesity Bridge perf stats.

## Features
- No need to sit and untar all the files, code will get only the requied bridge default files
- User will be able to understand Admission control Queue stats and Qos Queue principal details across all bridge nodes
- User can choose which stat they want to analyse the perf ( NFS|SMB|S3 for client ip / view stats with view id / disk stats diskId  / etc.)
- If backup/clone is happening only one one node, other inactive bridge nodes are droped during analysis 
- Time-based plotting
